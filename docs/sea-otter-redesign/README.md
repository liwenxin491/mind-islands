# Mind Islands Sea Otter Redesign

This folder contains review-first design explorations for the next major UI direction.

What is fixed across all concepts:
- There is only one avatar in the product: a sea otter
- The sea otter is self-externalization, not a pet
- First-person self-talk stays intact
- AI log assistant becomes the primary recording mode
- Islands move to a secondary surface instead of dominating the home screen

Files:
- `index.html`: visual concept board with 3 user-flow directions and 3 otter variants
- `styles.css`: shared styling for the concept board

Open locally:

```bash
open "/Volumes/Work/Codex/Mind Islands/docs/sea-otter-redesign/index.html"
```
